## Overview

The **Digital Literacy Portfolio** is a structured project developed as part of the Student Digital Ambassador role. This project showcases the development of key digital skills necessary for academic and professional growth. This project reflects the use of modern digital technologies, online platforms, and appropriate communication skills. This portfolio consists of tasks such as designing a digital awareness infographic, creating professional online profiles, learning about coding and collaboration platforms, composing formal emails, and learning about cyber safety through real-life scenarios.

## Module Summaries

### Module 1: Digital Literacy Awareness
For my first task as a Student Digital Ambassador, I created a high-impact one-page infographic so as to help in simplifying the complex concept of Digital Literacy for my fellow first-year batchmates. I used Canva to create an infographic that emphasizes three key pillars: 'Safe Internet Practices', 'Essential Academic Tools' and 'Professional Online Presence'.

### Module 2: Student Digital Portfolio
For my second task as a Student Digital Ambassador, I established a foundational Professional Identity on important sites like GitHub, LinkedIn, and Kaggle. This included creating a professional GitHub profile README to display my academic background and aspirations while doing the same on LinkedIn to connect with industry professionals. Kaggle, however, serves as a platform for accessing global competitions.

### Module 3: Coding & Collaboration Platforms
For my third task as a Student Digital Ambassador, I demonstrated my technical skills by completing a Python challenge on HackerRank. I also created a 'Digital Literacy Awareness Quiz' using Google Forms to engage and educate my batchmates on the same topic.

### Module 4: Professional Communication
For my fourth task as a Student Digital Ambassador, I drafted two formal emails, the former regarding an academic extension request while the latter being an internship inquiry. Furthermore, I created a Social Media Responsibility Checklist to assist students in developing a positive virtual footprint.

### Module 5: Cybercrime Awareness
For my fifth task as a Student Digital Ambassador, I researched about a realistic UPI/Online Payment Fraud case study while also creating an actionable "Stay Safe Online" prevention guide specifically for students in India.
